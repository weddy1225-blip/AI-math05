這是一個專為 **國小四年級學生** 設計的數學互動遊戲。透過「AI 偵探」的角色扮演，讓學生在限時挑戰中修正機器人錯誤的單位換算，從而強化對大數、長度、重量、面積與時間的量感。

👉 [**點擊這裡開始挑戰任務**](https://weddy1225-blip.github.io/AI-math05/)

---

## 🚀 遊戲特色
* **五大核心單元**：涵蓋大數、公里/公尺、公噸/公斤、面積/周長、時間換算。
* **隨機抽題系統**：內建 50 題題庫，每次遊戲自動從五大階段各抽一題，百萬種組合玩不膩！
* **即時回饋機制**：作答後立即顯示正確答案與解析，並在結算表提供「錯誤對照」，方便檢討。
* **兒童友善介面**：超大字體、繽紛色彩，並支援 `Enter` 鍵快速送出。

## 📖 遊戲規則
1. **觀察語病**：機器人 Code-Bot 會說出一串包含「錯誤單位換算」的話。
2. **找出正確數字**：忽略錯誤的資訊，在輸入框填入正確的換算數字。
3. **限時修正**：每題有 **30 秒** 的修正時間，考驗反應力與換算速度！

## 🛠️ 題目範圍 (符合小四課程大綱)
| 關卡類型 | 學習重點 |
| :--- | :--- |
| **大數密碼** | 兆、億、萬位的位值理解與讀寫。 |
| **距離測量** | 公里(km)與公尺(m)的進位換算。 |
| **重量挑戰** | 公噸(t)與公斤(kg)的換算。 |
| **面積周長** | 長方形與正方形的面積公式、平方公尺換算。 |
| **時間換算** | 24小時制、時/分/秒的單位轉化。 |
